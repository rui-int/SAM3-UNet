
#### How To Use

* Put the `SAM3UNet.py` into the [sam3 repo](https://github.com/facebookresearch/sam3). 
* Download the pretrained sam3 weights.
* Use the training/testing toolkit from [SAM2UNet](https://github.com/WZH0120/SAM2-UNet) or [SAM2UNeXT](https://github.com/WZH0120/SAM2-UNeXT).

#### Notice
* This repository is for proof of concept only, and the actual experimental results are not guaranteed.
